## Installation
```bash
pip install dlroWolleH
```

## Usage
```python
from dlroWolleH import HelloWorld

HelloWorld('print')
```

You can also try a different input:

```python
HelloWorld('input')
```

> The `HelloWorld('something')` will be executed as `something('Hello World!')`.
